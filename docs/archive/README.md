# Archive

Historical documents kept for reference only. **Not** a current source of truth — see
`docs/STATUS.md` (project state) and `docs/BACKLOG.md` (prioritized tasks) for that.

## Contents

| File | What it is |
|------|------------|
| `kiro-original-requirements.md` | The original 28-requirement EARS-style specification for the online poker platform, authored in the Kiro IDE. ~95% of it is implemented; the unfinished long-tail items are tracked as Phase 10 in `docs/BACKLOG.md`. |
| `kiro-original-design.md` | The original architecture/design document from the Kiro spec. The implemented architecture is documented in `docs/STATUS.md` and `CLAUDE.md`. |

## Provenance

These came from `.kiro/specs/online-poker-platform/`, the spec directory from when the project
was developed in the Kiro IDE. The Kiro IDE is no longer used. On 2026-06-26 the `.kiro`
directory was removed: every bug/test item in it was verified resolved, the only genuinely-open
features were migrated to `docs/BACKLOG.md` Phase 10, and these two reference docs were archived
here. The `kryky-test-fixes` spec (fully complete) and the testing-session logs were discarded.
